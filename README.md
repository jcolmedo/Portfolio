# Portfolio Profesional - Juan Carlos Olmedo

Bienvenido al repositorio de mi portfolio profesional. Este proyecto es una *landing page* o página web de un solo documento (*Single Page Application* en esencia) diseñada para mostrar mi experiencia técnica, liderazgo operativo y habilidades de análisis de datos.

## 🚀 Tecnologías y Diseño

El portfolio está construido priorizando el rendimiento, la accesibilidad y un diseño visual extremadamente moderno sin la dependencia de frameworks externos pesados:

- **HTML5 Semántico**: Estructura sólida y accesible para una mejor indexación (SEO).
- **CSS3 Personalizado (Vanilla CSS)**:
  - **Sistema de Diseño Premium**: Tema oscuro (*Dark Mode*) espacial que resalta con colores vibrantes (cian y violeta/azul).
  - **Glassmorphism**: Uso de paneles translúcidos (`backdrop-filter`) para dar un efecto de cristal esmerilado en las tarjetas y la barra de navegación.
  - **Micro-interacciones y Animaciones**: Efectos *hover* dinámicos, botones con *glow*, gradientes vivos y animaciones integradas vinculadas al scroll de la página utilizando JavaScript ligero (`IntersectionObserver`).
- **FontAwesome 6**: Integración de íconos sólidos vectoriales para lograr un lenguaje visual profesional y coherente en todo el proyecto.
- **JavaScript Moderno**: Manejo ligero de eventos para *smooth scrolling* y la aparición difuminada (*fade-in*) de los elementos al hacer *scroll*.

## 📁 Estructura del Proyecto

El proyecto está diseñado para ser limpio, modular (en CSS) y fácil de desplegar.

```plaintext
/
├── index.html       # Documento principal con la estructura semántica y el contenido.
├── styles.css       # Archivo de estilos (variables globales, layout, animaciones y glassmorphism).
├── img/             # Carpeta de recursos gráficos.
│   └── Foto_perfil.png # Fotografía o avatar del usuario.
└── README.md        # Documentación del proyecto.
```

## 📝 Secciones Principales

1. **Hero Section (Inicio)**: Presentación principal y llamado a la acción (*Call to Action*).
2. **Sobre Mí**: Resumen de enfoque técnico, valores profesionales y estadísticas rápidas.
3. **Experiencia**: Línea de tiempo que detalla el historial profesional y logros alcanzados a nivel IT y analítico.
4. **Especialidades (Servicios)**: Descripción de las áreas críticas de especialidad (Automatización, Datos, Liderazgo, etc.).
5. **Stack Tecnológico**: Etiquetas (*badges*) visuales de los lenguajes, frameworks y herramientas dominadas.
6. **Proyectos Destacados**: Tarjetas con los casos de éxito, ilustrando el impacto técnico y funcional en diferentes organizaciones.
7. **Contacto**: Información y enlaces directos para conexión profesional.

## 💻 Desarrollo Local y Despliegue

Dado que es un sitio web estático (archivos HTML y CSS planos), no requiere un entorno de servidor complejo ni dependencias de Node.js.

### ¿Cómo correr el proyecto localmente?
Basta con abrir el archivo `index.html` en cualquier navegador web moderno (Chrome, Firefox, Edge, Safari). Alternativamente, si usas VS Code, puedes utilizar la extensión **Live Server** para una recarga automática durante el desarrollo.

### Recomendaciones de Despliegue
Este repositorio está listo para ser desplegado fácilmente y sin costo a través de cualquier servicio de hosting estático, siendo los más recomendables:
- [GitHub Pages](https://pages.github.com/) (Ideal si este repositorio es parte de tu cuenta y quieres ligarlo a tu perfil).
- [Vercel](https://vercel.com/) o [Netlify](https://www.netlify.com/).
